# Проект маппера для автотестов
```
0. Установить библиотеки из requirements.txt
1. Скачать вебдрайвер и положить в папку drivers в корень проекта
2. Запустить StartWindow.py
3. Нажать кнопку Маппер
4. Сделать активной (в фокусе) форму PyQT, а не webdriver
5. Для запуска маппера нажать на клавишу R
6. Для нахождения атрибутов вебэлемента нажать на F4
```
