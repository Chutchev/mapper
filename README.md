# Проект маппера для автотестов
```
0. Установить библиотеки из requirements.txt
1. Скачать вебдрайвер и положить в папку drivers в корень проекта
2. Запустить autotest_script.py
3. Для запуска маппера нажать на клавишу R
4. Для нахождения атрибутов вебэлемента нажать на F4
5. Для завершения работы скрипта нажать на Q
```
